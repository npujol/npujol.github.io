```
# Grammatik: Vorgangspassiv

Das Vorgangspassiv beschreibt eine Handlung oder einen Prozess, der an einem Subjekt stattfindet. Im Fokus steht die Aktion selbst, nicht der Handelnde (Agens).
```

Es wird mit „werden“ + Partizip II gebildet und kann in allen Zeitformen stehen.

## Verwendung des Vorgangspassivs

1. Agens ist unbekannt/unwichtig:

- *Das Fenster wurde geöffnet.* (Wer es öffnete, ist irrelevant.)

1. Fokus auf die Handlung, nicht auf den Handelnden:

- *Die Straße wird repariert.* (Wichtig: die Reparatur, nicht der Arbeiter.)

1. In wissenschaftlichen/technischen Texten:

- *Die Proben wurden analysiert.* (Objektivität steht im Vordergrund.)

### Struktur des Vorgangspassivs

| Zeitform | Formel | Beispiel (Aktiv → Passiv) |
|-------------------|---------------------------------|------------------------------------------------|
| Präsens | werden + Partizip II | *Sie baut das Haus.* → *Das Haus wird gebaut.* |
| Präteritum | wurden + Partizip II | *Sie baute das Haus.* → *Das Haus wurde gebaut.* |
| Perfekt | sein + Partizip II + worden | *Sie hat das Haus gebaut.* → *Das Haus ist gebaut worden.* |
| Plusquamperfekt| waren + Partizip II + worden | *Sie hatte das Haus gebaut.* → *Das Haus war gebaut worden.* |
| Futur I | werden + Partizip II + werden | *Sie wird das Haus bauen.* → *Das Haus wird gebaut werden.* |

### Aktiv vs. Vorgangspassiv – Vergleich

| Aktivsatz | Vorgangspassiv | Fokus |
|----------------------------------|------------------------------------|--------------------------------|
| *Der Mechaniker repariert das Auto.* | *Das Auto wird repariert.* | Handlung (Reparatur) |
| *Die Schüler lesen den Text.* | *Der Text wird gelesen.* | Prozess (Lesen) |

### Agens im Vorgangspassiv

Der Handelnde kann mit „von“ (Personen) oder „durch“ (Instrumente/Ursachen) genannt werden, ist aber oft weggelassen:

- *Das Buch wird vom Autor signiert.* (Person)
- *Die Stadt wurde durch einen Sturm zerstört.* (Ursache)

### Vorgangspassiv vs. Zustandspassiv

| | Vorgangspassiv | Zustandspassiv |
|--------------------------|-------------------------------------|-------------------------------------|
| Bedeutung | Prozess/Aktion | Ergebnis/Zustand |
| Hilfsverb | werden | sein |
| Beispiel | *Das Haus wird gebaut.* (Aktion) | *Das Haus ist gebaut.* (Fertiger Zustand) |
| Zeitformen | Alle Zeitformen möglich | Nur Präsens/Präteritum/Perfekt |
