```
# Grammatik: Nominalisierung & Nominalstil

Im Nominalstil, der typisch für die Wissenschaftssprache ist, tragen Nomen die Hauptbedeutung eines Satzes. Dies wird häufig genutzt, um Informationen verkürzt darzustellen.
```

## Nominalisierung und Verbalisierung von Präpositionalergänzungen

Präpositionalergänzungen können in einen dass-Satz oder Infinitivsatz umgeformt werden. Einen Infinitivsatz kann man nur bilden, wenn das Subjekt des Nebensatzes mit einer Ergänzung des Hauptsatzes identisch oder das Subjekt des Nebensatzes man ist.
Bei der Umformung wird die Präposition zu einem Präpositionaladverb im Hauptsatz. Bei vielen Verben kann das Präpositionaladverb weggelassen werden.

| Nominalform | Verbalform |
| ---------------------------------------------------------------- | --------------------------------------------------------------------- |
| Brainstorming setzt man für die Entwicklung neuer Ideen ein. | Brainstorming setzt man dafür ein, neue Ideen zu entwickeln. |
| Ihr Umfeld wird sich über Ihre Kreativität freuen. | Ihr Umfeld wird sich (darüber) freuen, dass Sie kreativ sind. |

## Nominalisierung von Verben

- Nominalisierung des Infinitivs (immer n.)
- Verwendung des Verbstamms mit der Endung *-ung* bei den meisten schwachen Verben (immer f.)
- Verwendung des reinen Verbstamms (teilweise mit Vokalwechsel) (meist m.)
- Verwendung des Verbstamms mit der Endung *-e* (meist f.)
- Verwendung fester Formen
- Verwendung der Endung *-tion* bei Verben auf *-ieren*

| Verb | Nomen |
| ------------- | ---------------- |
| gehen | das Gehen |
| wirken | die Wirkung |
| lösen | die Lösung |

## Nominalisierung von Adjektiven

- Verwendung des Wortstamms mit der Endung *-e* (für Personen und Objekte)
- Verwendung der Endung *-keit* bei Adjektiven auf *-ig* (immer f.)
- Verwendung fester Formen

| Adjektiv | Nomen |
|----------|-------|
| alt | der/die Alte |
| schön | das Schöne |
| abhängig | die Abhängigkeit |
| alt | das Alter |
| jung | die Jugend |

## Vom Verbalstil zum Nominalstil: Umwandlung von adverbialen Nebensätzen

| Nebensatz | Konjunktion | Präposition + Nomen |
|-----------|------------|----------------------|
| kausal (Grund) | weil, da | aufgrund / wegen / infolge + Gen. |
| konzessiv (Einschränkung) | obwohl | trotz + Gen. |
| modal (Art und Weise) | indem | durch + Akk. |
| temporal (Zeitangabe) | sobald | mit + Dat. |
| | während, solange | während + Gen. |
| | bevor / nachdem | vor + Dat. / nach + Dat. |
| final (Zweck, Ziel) | um … zu + Inf. | zu + Dat., zwecks (+ Gen., im Sg. meist ohne Art.), für + Akk. |
| | damit | |
| konditional (Bedingung) | wenn, falls | bei + Dativ |

### Ursache vs. Wirkung: Begriffe kategorisieren

| Ursache (Grund/Auslöser) | Wirkung (Folge/Ergebnis) |
| ---------------------------- | ---------------------------- |
| der Auslöser | die Folge |
| der Grund | die Konsequenz |
| die Voraussetzung | das Resultat |

Erklärung:

- Ursache: Begriffe, die den Anlass oder Grund für etwas beschreiben (z. B. *Auslöser*, *Grund*).
- Wirkung: Begriffe, die das Ergebnis oder die Folge einer Handlung beschreiben (z. B. *Folge*, *Resultat*).
- *Die Voraussetzung* ist eine Bedingung, die erfüllt sein muss, damit etwas passiert (ähnlich einer Ursache).

### Synonyme um Folgen zu sagen

| Synonyme | Unpassendes Wort | Grund |
| --------------------------------- | ---------------- | --------------------------------------------------------- |
| bewirken, verursachen | einwirken | *Einwirken* bedeutet "beeinflussen", nicht "verursachen". |
| auslösen, führen zu | ablösen | *Ablösen* bedeutet "ersetzen", nicht "auslösen". |
| mit sich nehmen, mit sich bringen | führen nach | *Führen nach* ist falsch; korrekt: führen zu. |

### Sätze umformulieren: Nominalstil mit Ursache-Wirkung

Verwende Präpositionen (aufgrund, wegen, durch) und Nominalisierungen (Endung *-ung*, *-tion*).

#### Beispiel 1

Original:
*Aufgrund des technischen Fortschritts kommt es zu einem Wandel des Arbeitsmarktes.*

Umformulierungen (Nominalstil):

- Der technische Fortschritt ist der Auslöser für den Wandel des Arbeitsmarktes.
- Die Weiterentwicklung der Technik führt zur Veränderung des Arbeitsmarktes.

#### Beispiel 2

Original:
*Wegen der ersten technischen Revolution kam es zu einer Landflucht.*

Umformulierung:

- Die erste technische Revolution löste eine Landflucht aus.

#### Beispiel 3

Original:
*Dank der neuesten Technologien wurde das Arbeitsleben erleichtert.*

Umformulierung:

- Die neuesten Technologien bewirken eine Erleichterung des Arbeitslebens.

#### Beispiel 4

Original:
*Durch intensive Forschung und Entwicklung können Science-Fiction-Visionen wohl bald verwirklicht werden.*

Umformulierung:

- Intensive Forschung und Entwicklung ermöglichen die Verwirklichung von Science-Fiction-Visionen.

### Wichtige Präpositionen für Ursache-Wirkung

| Ursache | Präposition | Beispiel |
| ------- | --------------- | ------------------------------- |
| Grund | aufgrund, wegen | *Aufgrund des Fortschritts ...* |
| Mittel | durch | *Durch Forschung ...* |
| Zweck | zu, zwecks | *Zwecks Verbesserung ...* |
