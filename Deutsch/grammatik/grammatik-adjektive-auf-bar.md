\# Grammatik: Adjektive auf -bar

```
## Regeln zur Bildung von Adjektiven auf -bar

Adjektive auf - bar werden meist aus Verben gebildet und bedeuten, dass etwas möglich oder machbar ist.
```

Sie ersetzen oft Passivsätze mit *können* (z. B. *kann gemacht werden* → *ist machbar*).

### 1. Grundregel: Verb-Stamm + -bar

| Verb | Adjektiv auf -bar | Bedeutung |
| --------------- | ------------------- | ----------------------- |
| lesen → | lesbar | *kann gelesen werden* |
| öffnen → | öffnbar | *kann geöffnet werden* |
| reparieren→ | reparierbar | *kann repariert werden* |
| vergessen → | vergessbar <br> | kann vergessbar werden |

### 2. Ausnahmen & Besonderheiten

- Verben auf *-ieren*: Das -e- wird vor *-bar* entfernt:
  *diskutieren* → diskutierbar

- Verben mit trennbaren Präfixen:
  *aussprechen* → aussprechbar

*vergessen* → vergessbar(kein Umlaut)
*brechen* → brechbar (kein Umlaut)

## Verwendung im Satz – Beispiele

### Passiv mit „können“ vs. Adjektiv auf -bar

| Passiv mit „können“ | Adjektiv auf -bar |
|---------------------------------|------------------------------|
| *Das Wasser kann getrunken werden.* | *Das Wasser ist trinkbar.* |
| *Das Problem kann gelöst werden.* | *Das Problem ist lösbar.* |
| *Der Text kann übersetzt werden.* | *Der Text ist übersetzbar.* |

## Wichtige Tipps & Fehlerquellen

1. Nicht alle Verben bilden -bar!

- Nur transitive Verben (Verben mit Akkusativobjekt) bilden -bar:
  *sehen* → sichtbar ✔️
  *gehen* → ❌ (kein Akkusativobjekt → kein *gehbar*)

- Intransitive Verben funktionieren nicht:
  *schlafen* → ❌ *schlaftbar*
  *laufen* → ❌ *laufbar*

1. Achtung vor falschen Bedeutungen!

- *Denkbar* bedeutet nicht „kann gedacht werden“, sondern „vorstellbar“:
  *Das ist eine denkbare Lösung.* = *Diese Lösung ist vorstellbar.*

1. Keine Verwechslung mit Partizipien:

- *geschlossen* (Partizip) vs. *schließbar* (Adjektiv auf -bar).
  *Die Tür ist geschlossen.* (Zustand)
  *Die Tür ist schließbar.* (Möglichkeit)
